ALUMA - Assistente de Lógica de Uso para Memória e Aprendizado

HTML CSS JavaScript

O projeto de Trabalho de Conclusão de Curso (TCC) desenvolvido na Fundação O Pão dos Pobres apresenta a ALUMA (Assistente de Lógica de Uso para Memória e Aprendizado), uma Inteligência Artificial voltada à área da educação, com o objetivo de democratizar o acesso ao conhecimento, facilitar o aprendizado e auxiliar professores, alunos e instituições de ensino em geral.

Objetivo:

A ALUMA foi projetada para ser uma ferramenta de apoio didático e pedagógico, oferecendo uma interface interativa capaz de responder dúvidas, revisar conteúdos, sugerir atividades e reforçar a memorização por meio de uma lógica personalizada de uso. A IA é adaptável para escolas, cursos livres, ONGs educacionais e outras instituições que busquem inovação no processo de ensino-aprendizagem.

Ada Lovelace: A Mãe da Programação

HTML CSS JavaScript

Este projeto foi desenvolvido em grupo como parte das atividades da disciplina de Desenvolvimento Pessoal e Social (DPS) na Fundação O Pão dos Pobres. O trabalho teve como objetivo resgatar e valorizar a trajetória de Ada Lovelace, figura histórica reconhecida como a primeira programadora da história da computação.

Mais do que um simples estudo biográfico, o projeto buscou explorar o impacto da contribuição de Ada Lovelace no cenário atual da tecnologia, especialmente na formação de jovens em áreas da programação.

Objetivo:

Destacar a importância de Ada Lovelace como pioneira na área da computação e refletir sobre o papel das mulheres na ciência e na tecnologia, promovendo a valorização da diversidade e o empoderamento feminino nas áreas técnicas.

Diferenciais do Projeto:

  • Pesquisar a vida e as conquistas de Ada Lovelace.
  • Compreender o contexto histórico e científico em que ela viveu.
  • Discutir a invisibilidade feminina nas áreas tecnológicas ao longo da história.
  • Relacionar a contribuição de Ada com o cenário atual da programação.

Outubro Rosa Conscientiza.r

HTML CSS JavaScript

Este projeto foi desenvolvido como parte das atividades práticas na Fundação O Pão dos Pobres, com o objetivo de criar um site informativo voltado à campanha Outubro Rosa — um movimento internacional de conscientização para a prevenção e o diagnóstico precoce do câncer de mama.

Objetivo:

Utilizar a tecnologia como ferramenta de impacto social, promovendo o acesso à informação de forma clara, acessível e visualmente atrativa, especialmente para públicos que muitas vezes têm pouco contato com ações de saúde preventiva.

Diferenciais do Projeto:

  • Cores temáticas e ícones que reforçam a identidade do Outubro Rosa.
  • Navegação simples e acessível para públicos de todas as idades.
  • Linguagem inclusiva, objetiva e sensível ao tema.
  • Versão mobile responsiva, permitindo o acesso pelo celular.

Lista de Tarefas

HTML CSS JavaScript

Este projeto foi desenvolvido como parte do curso de Desenvolvimento Web Full Stack no Instituto da Oportunidade Social (IOS). Nele, criei uma lista de tarefas interativa e, como desafio pessoal, implementei pela primeira vez o modo escuro (dark mode), com a intenção de explorar novos conceitos de estilização com CSS.

Objetivo:

A proposta principal era fazer uma atividade do curso com foco em JavaScript para desenvolver uma interface interativa que permitisse ao usuário adicionar, marcar como concluída e remover tarefas — tudo isso com a opção de alternar entre o modo claro e escuro, proporcionando uma experiência visual personalizada.

Funcionalidades principais:

  • Adicionar tarefas dinamicamente.
  • Marcar tarefas como concluídas com apenas um clique.
  • Remover tarefas da lista.
  • Alternar entre modo claro e escuro (dark mode) com suavidade.
  • Animação de background RGB dinâmico, dando um toque moderno ao visual.
